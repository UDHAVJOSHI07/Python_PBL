📌 Project Title

Heart Disease Prediction using Machine Learning

👥 Team Members

- Udhav Joshi (Team Lead)
- Devansh Jain

📖 Synopsis

This project aims to predict the likelihood of heart disease using machine learning techniques.
We use patient health data such as age, blood pressure, cholesterol levels, etc.,
to train models like Linear Regression and KNN.

🎯 Objective

- To analyze health data
- To build a prediction model
- To detect heart disease at an early stage

⚙️ Technologies Used

- Python
- Machine Learning
- NumPy
- Pandas
- Scikit-learn
